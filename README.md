# SanGuoSha Final Project

This is a PyCharm-ready single-player SanGuoSha-style card game. The UI is designed like an online SanGuoSha battle table, with local card and character assets.

## How to Run

1. Open this folder in PyCharm.
2. Choose the `Run_Game` run configuration in the top-right corner.
3. Click the green Run button.

You can also double-click `run_game.bat`.

## Features

- Player vs computer.
- Online SanGuoSha-style table layout.
- Real card images for `杀`, `闪`, `桃`, `无中生有`, `过河拆桥`, and `决斗`.
- SanGuoSha-style character images.
- Basic rules: drawing cards, playing cards, Strike/Dodge response, healing, Duel, Dismantle, and Insight.
- Simple computer AI.

## Assets

The assets are downloaded into `assets/sgs`, so the game does not need internet access while running.
